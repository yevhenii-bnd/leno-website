## Project Review

This project was created as a **learning exercise** while completing the Udemy course  
[Modern HTML & CSS From The Beginning 2.0](https://www.udemy.com/course/modern-html-css-from-the-beginning).

The goal of the project was to practice building a modern landing page using **semantic HTML, structured CSS architecture, and basic JavaScript interactions**.

### Key aspects

- **CSS architecture:** structured using the **BEM (Block Element Modifier)** methodology  
- **Layout:** responsive landing page built with semantic HTML  
- **JavaScript:** basic DOM manipulation and UI interactions

### Implemented JavaScript features

- Mobile navigation toggle  
- Video modal (open / close logic)  
- Dynamic navbar styling on scroll  

### Tech stack

- HTML5  
- CSS3  
- BEM methodology  
- Vanilla JavaScript (DOM API)
